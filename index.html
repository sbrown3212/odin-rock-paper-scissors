Rock, Paper, Scissors

To play this game, you must be able to view the console in developer tools.

To open console:

Mac OS:

Safari: "Command" + "Option" + "C"

Chrome: "Command" + "Option" + "J"

Firefox: "Command" + "Option" + "K"

Windows

Edge: "Control" + "Shift" + "J"

Chrome: "Control" + "Shift" + "J"

Firefox: "Control" + "Shift" + "K"
